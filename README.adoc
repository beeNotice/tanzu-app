= Tanzu App

== Prerequisites


Weavefront account 
* https://wavefront.surf


== Launch

[source, sh]
----
# Run app
./mvnw spring-boot:run

# Build image
./mvnw spring-boot:build-image
----

== Access

* http://localhost:8080
* http://localhost:8080/h2-console
